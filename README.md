# RyzenTU - A AMD Mobile Ryzen Tweak Tool

一个调整 AMD Mobile Ryzen CPU 功率限制的小工具，支持 Ryzen 3 - Ryzen 7。 

所有的调整重启后失效，使用时注意设备供电功率是否足够
